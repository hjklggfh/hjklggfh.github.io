# Pixel Assets

Place pixel-art image files anywhere under this folder, or use one of the root-level asset folders listed in the project README. The engine scans common image formats with `import.meta.glob` and classifies them by filename.

Recommended filename hints:

- `grass`, `field`, `meadow`
- `water`, `river`, `lake`
- `tree`, `forest`
- `mountain`, `cliff`, `rock`
- `road`, `path`, `dirt`
- `bridge`
- `house`, `roof`, `wall`
- `fence`, `lantern`
- `npc`, `character`

The game also supports browser directory import through the Assets button when File System Access API is available.
