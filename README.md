# 🛡️ 保卫萝卜 — 2D 塔防网页游戏

纯 HTML/CSS/JavaScript 实现的俯视视角塔防游戏，类似《保卫萝卜》玩法。怪物沿贝塞尔路径前进，玩家在草坪上建造炮塔阻止怪物到达终点。

## 快速启动

```bash
cd 项目目录
python -m http.server 8080
# 浏览器打开 http://localhost:8080
```

或者直接用浏览器打开 `index.html`。

## 玩法

### 操作方式
| 操作 | 效果 |
|---|---|
| 点击绿色平台 | 打开建造面板 |
| 选择炮塔并点击 | 消耗金币建造 |
| 点击已有炮塔 | 打开升级/维修/售出面板 |
| 右键 / 点击空地 | 取消选择 |
| ⏸ 暂停按钮 | 冻结战斗，仍可建造/升级 |
| 右上角速度按钮 | 1x / 2x 切换 |

### 模式
- **普通模式**：8 波预设关卡，手动推进
- **无尽模式**：无限波次，Boss 每 5 波，点击"开始无尽"启动

### 胜负条件
- 萝卜初始 10 点 HP，显示在左上角 ❤️
- 每漏掉一个怪扣 1~5 点（视怪物类型）
- HP 归零 → 失败；消灭全部波次 → 胜利

## 炮塔 (6 种，最高 10 级)

| 炮塔 | 特点 | 基础HP | 费用 |
|---|---|---|---|
| 🔵 瓶子炮 | 单体+溅射50px(40伤害) | 40 | 100 |
| ❄️ 冰冻星 | 群体减速60% · 溅射80px | 30 | 120 |
| 🌻 太阳花 | 自身200px脉冲AOE | 30 | 150 |
| 🚀 火箭炮 | 高伤爆炸 · 溅射100px | 40 | 200 |
| ⚡ 激光炮 | 300射程激光束 · 破甲 · 优先Boss | 30 | 300 |
| 💥 火炮 | AOE150px · 灼伤10/秒 | 25 | 350 |

## 调整游戏数值

编辑 `js/data/data.js`，修改对应字段后刷新浏览器。例如：

```js
// 修改瓶子炮 Lv1 伤害
bottleCannon: {
    levels: [
        { range: 120, fireRate: 1.5, damage: 20, ... },  // 改这个 20
        ...
    ]
}
```

字段含义：`range` 射程 · `fireRate` 攻速(次/秒) · `damage` 伤害 · `upgradeCost` 升级费 · `sellValue` 售出价

## 技术要点

- **贝塞尔路径**：三次贝塞尔曲线，支持直线段（控制点放 1/3 和 2/3 处即为直线）
- **攻击优先级**：默认瞄准离终点最近的敌人（progress 最高），激光优先 Boss
- **视线遮挡**：障碍物对炮塔视线进行圆-线段相交检测
- **暂停机制**：dt 置零冻结实体更新，UI 事件不依赖帧循环
- **图鉴同步**：图鉴数据直接从 `GameConfig` 动态读取，修改 `data.js` 自动同步
